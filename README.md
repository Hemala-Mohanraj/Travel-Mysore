# Travel Mysore

## Project Description

Travel Mysore is a static tourism website developed using HTML, CSS, and JavaScript. The website provides visitors with information about Mysore, including its heritage, tourist attractions, hotels, and cultural significance. It features a clean user interface with responsive layouts, image cards, a navigation bar, and a modal popup for an enhanced user experience.

## Features

- Home page with a hero section
- Navigation bar with links to different sections
- Information about Mysore
- Tourist attractions displayed using cards
- Responsive Grid and Flexbox layouts
- Modal popup window
- Image gallery and cards
- Styled buttons with hover effects
- Footer section
- Simple and user-friendly interface

## JavaScript Functionality

The project uses JavaScript to control a modal popup.

Functions:

- `openModal()` – Displays the modal window.
- `closeModal()` – Hides the modal window.

The modal is initially hidden using `display: none` and becomes visible by changing the display property to `block`.
